These text files provide guides that you can use to adapt chatbots to hepl you play "Settlers of Mars" games.

These guides can be given to chat bots such as OpenAI GPT4.0, OpenAI DALLE, OpenAI Game Time, Perplexicty.ai, Ora.ai, You.ai, nightcafe.ai to "train" AI chat bots to help you navigate the game.

If you fine-tune your own Settlers of Mars Chat bot script please feel free to share it here :-)
